Detailed Description

An implementation of Fulfillment for a crypto-condition fulfillment of type "PREIMAGE-SHA-256" based upon a preimage and the SHA-256 hash function.

See also
"https://datatracker.ietf.org/doc/draft-thomas-crypto-conditions/"
