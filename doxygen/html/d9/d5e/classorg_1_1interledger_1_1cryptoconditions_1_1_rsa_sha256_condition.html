org.interledger.cryptoconditions.RsaSha256Condition.RsaSha256Condition ( final RSAPublicKey  key)

Constructs an instance of the condition.

Development Note: This constructor is not package-private because it may be desirable for a party who only has a public-key to be able to create an RSA-SHA-256 Condition. For example, it might be useful to create a condition that some other party, who does have access to the RSA private key, to fulfill the condition.

Parameters
keyThe RSA public key associated with the condition.

Constructs an instance of this condition from it's corresponding parts.

This constructor should be primarily used by Codecs, whereas developers should, in general, use the RsaSha256Condition(RSAPublicKey) constructor, or else create an actual fulfillment via RsaSha256Fulfillment#RsaSha256Fulfillment(RSAPublicKey, byte[]) and then generate a condition from that object.

Parameters
costThe calculated cost of the condition.
fingerprintThe calculated fingerprint for the condition.
