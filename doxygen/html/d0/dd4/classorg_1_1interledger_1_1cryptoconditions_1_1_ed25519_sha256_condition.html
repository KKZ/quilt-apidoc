org.interledger.cryptoconditions.Ed25519Sha256Condition.Ed25519Sha256Condition ( final byte []  fingerprint)

Constructs an instance of the condition with the given fingerprint and cost.

This constructor should be primarily used by CODECs, whereas developers should, in general, use the Ed25519Sha256Condition(EdDSAPublicKey) constructor, or else create an actual fulfillment via byte[]) and then generate a condition from that object.

Parameters
fingerprintThe binary representation of the fingerprint for this condition.
