Detailed Description

An abstract implementation of VoidHandler for allowing callers to specify logic based upon an unknown result type extending InterledgerPacket. This class can be used in the following manner:


final InterledgerPacket decodedPacket = context.read(asn1OerPaymentBytes);
new AbstractVoidHandler() {
  protected void handle(final InterledgerPayment interledgerPayment) {
     // ... do handling here.
  }
}.execute(decodedPacket); // be sure to call .execute!
