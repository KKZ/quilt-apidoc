Detailed Description

A response to a quote request with liquidity information regarding transfers between the current ledger and the destination account. This information is sufficient to locally quote any amount until the curve expires.

◆ getAppliesToPrefix()

InterledgerAddress org.interledger.ilqp.QuoteLiquidityResponse.getAppliesToPrefix ( )

A common address prefix of all addresses for which the above liquidity curve applies. If the curve only applies to the destination account (see #getDestinationAccount()) of the corresponding quote request, then this value will be equal to that address. If the curve applies to other accounts with a certain prefix, then this value will be set to that prefix.

For more on ILP Address Prefixes, see InterledgerAddress.

Returns
An instance of InterledgerAddress.

Instant org.interledger.ilqp.QuoteLiquidityResponse.getExpiresAt ( )

Maximum time where the connector (and any connectors after it) expects to be able to honor this liquidity curve. Note that a quote in ILP is non-committal, meaning that the liquidity is only likely to be available – but not reserved – and therefore not guaranteed.

◆ getLiquidityCurve()

LiquidityCurve org.interledger.ilqp.QuoteLiquidityResponse.getLiquidityCurve ( )

A series of exchange rates that can be plotted to assemble a "curve" of liquidity representing the amount that one currency can be exchanged for another.

For example, if a liquidity curve contains the rate [0,0] and [10,20], then there is a linear path of rates for which one currency can be exchange for another. To illustrate, it can be assumed that [5,10] exists on this curve.

Returns
A List of type {link ExchangeRate}.
