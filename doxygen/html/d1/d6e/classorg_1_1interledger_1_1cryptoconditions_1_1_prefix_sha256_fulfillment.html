byte [] org.interledger.cryptoconditions.PrefixSha256Fulfillment.getPrefix ( )

Accessor for the prefix as an array of bytes.

Returns
A byte array containing the prefix for this fulfillment.
Deprecated:
Java 8 does not have the concept of an immutable byte array, so this method allows external callers to accidentally or intentionally mute the prefix. As such, this method may be removed in a future version. Prefer getPrefixBase64Url() instead.
