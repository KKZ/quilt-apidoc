byte []  nonce,
byte []  data 
)
static

Encrypts a block of data using the encryption scheme specific in the PSK RFC and returns the encrypted data and the authentication.

NOTE: May throw an InvalidKeyException if the Java Cryptography Extension (JCE) Unlimited Strength Jurisdiction Policy Files are not installed.

See also
http://www.oracle.com/technetwork/java/javase/downloads/jce8-download-2133166.html
Parameters
keyThe AES encryption key.
nonceThe 16 byte nonce to use as an IV.
dataThe data to encrypt.
Returns
The encrypted data and its accompanying GCM authentication tag.
Exceptions
RuntimeExceptionif there is an error encrypting the data
